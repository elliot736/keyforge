## Installation

```bash
npm install @keyforge/sdk
```

## Quick Start

Register the KeyForge plugin to validate API keys on incoming requests. The verified key context is available on `request.keyforge`.

```ts
import Fastify from 'fastify';
import { KeyForge } from '@keyforge/sdk';
import { keyforgePlugin } from '@keyforge/sdk/middleware/fastify';

const fastify = Fastify();
const kf = new KeyForge({ apiUrl: '...', rootKey: '...' });

fastify.register(keyforgePlugin, { keyforge: kf });

fastify.get('/api/completions', async (request, reply) => {
  const { keyId, ownerId } = request.keyforge!;
  return { message: `Hello, ${ownerId}` };
});

fastify.listen({ port: 3000 });
```

## Options

Pass options as the second argument to `fastify.register`:

| Option | Type | Description |
| --- | --- | --- |
| `keyforge` | `KeyForge` | **Required.** A configured KeyForge SDK instance. |
| `prefix` | `string` | Route prefix to protect. Defaults to `/` (all routes). |
| `extractKey` | `(request: FastifyRequest) => string \| undefined` | Custom function to extract the API key. Defaults to `Authorization: Bearer <key>`. |
| `onError` | `(error: Error, request: FastifyRequest, reply: FastifyReply) => void` | Custom error handler. Defaults to a `401` JSON response. |

### Scoping to a Prefix

Protect only a subset of routes by specifying a prefix:

```ts
fastify.register(keyforgePlugin, {
  keyforge: kf,
  prefix: '/api',
});

// This route is protected
fastify.get('/api/data', async (request) => {
  return { owner: request.keyforge!.ownerId };
});

// This route is NOT protected
fastify.get('/health', async () => {
  return { status: 'ok' };
});
```

### Custom Key Extraction

```ts
fastify.register(keyforgePlugin, {
  keyforge: kf,
  extractKey: (request) => request.headers['x-api-key'] as string,
});
```

## Rate Limit Headers

When rate limiting is configured on a key, the plugin sets the following headers:

| Header | Description |
| --- | --- |
| `X-RateLimit-Limit` | Maximum requests allowed in the current window. |
| `X-RateLimit-Remaining` | Requests remaining in the current window. |
| `X-RateLimit-Reset` | Unix timestamp when the window resets. |
| `Retry-After` | Seconds to wait. Present only on `429` responses. |

## TypeScript

Add type augmentation for full IntelliSense:

```ts
declare module 'fastify' {
  interface FastifyRequest {
    keyforge?: {
      keyId: string;
      ownerId: string;
      scopes: string[];
      meta: Record<string, unknown>;
      rateLimit?: {
        limit: number;
        remaining: number;
        reset: number;
      };
    };
  }
}
```

## Decorators

The plugin registers a `keyforge` decorator on the Fastify instance, giving you direct access to the SDK from any handler:

```ts
fastify.get('/admin/keys', async (request, reply) => {
  const keys = await fastify.keyforge.keys.list({ ownerId: 'user_123' });
  return keys;
});
```
